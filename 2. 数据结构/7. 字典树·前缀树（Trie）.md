#Trie #字典树

## 普通 Trie（字符串前缀）

### 感性理解

假设字符串里面只有 a 和 b 两种字符。

从左到右遍历字符串，把 a 视作往左走，把 b 视作往右走。怎么记录路径？怎么区分不同的路径？用二叉树记录。

- insert：例如先插入字符串 aab，相当于生成了一条移动方向为「左-左-右」的路径。标记最后一个节点为终止节点。再插入字符串 aabb，相当于生成了一条移动方向为「左-左-右-右」的路径。标记最后一个节点为终止节点。
- search：例如查找字符串 aab，相当于查找二叉树中是否存在一条移动方向为「左-左-右」的路径，且最后一个节点是终止节点。
- startsWith：例如查找前缀 aa，相当于查找二叉树中是否存在一条移动方向为「左-左」的路径，无其他要求。

推广到 26 种字母，其实就是一棵 26 叉树，对于 26 叉树的每个节点，可以用哈希表，或者长为 26 的数组来存储子节点。
### 算法实现

- **初始化**：创建一棵 26 叉树，一开始只有一个根节点 `root`。26 叉树的每个节点包含一个长度为 26 的子节点列表 `son`，以及一个布尔值 `end`，表示是否为终止节点。

- **insert**：
  1. 遍历字符串 `word`，同时用一个变量 `cur` 表示当前在 26 叉树的哪个节点，初始值为 `root`。
  2. 如果 `word[i]` 不是 `cur` 的子节点，那么创建一个新的节点 `node` 作为 `cur` 的子节点。如果 `word[i] = a`，把 `node` 记录到 `cur` 的 `son[0]` 中。如果 `word[i] = b`，把 `node` 记录到 `cur` 的 `son[1]` 中，依此类推。
  3. 更新 `cur` 为子列表中的相应节点。
  4. 遍历结束，把 `cur` 的 `end` 标记为 `true`。

- **search 和 startsWith 可以复用同一个函数 find**：
  1. 遍历字符串 `word`，同时用一个变量 `cur` 表示当前在 26 叉树的哪个节点，初始值为 `root`。
  2. 如果 `word[i]` 不是 `cur` 的子节点，返回 0。`search` 和 `startsWith` 收到 0 之后返回 `false`。
  3. 更新 `cur` 为子列表中的相应节点。
  4. 遍历结束，如果 `cur` 的 `end` 是 `false`，返回 1，否则返回 2。`search` 如果收到的是 2，返回 `true`，否则返回 `false`。`startsWith` 如果收到的是非 0 数字，返回 `true`，否则返回 `false`。

```cpp
struct Node {
    Node* son[26]{};
    bool end = false;
};

class Trie {
    Node* root = new Node();

    int find(string word) {
        Node* cur = root;
        for (auto c : word) {
            c -= 'a';
            if (cur->son[c] == nullptr) {
                return 0;
            }
            cur = cur->son[c];
        }
        return cur->end ? 2 : 1;
    }

    void destroy(Node* node) {
        if (node == nullptr)
            return;
        for (auto son : node->son)
            destroy(son);
        delete node;
    }

public:
    ~Trie() { destroy(root); }
    Trie() {}

    void insert(string word) {
        Node* cur = root;
        for (auto c : word) {
            c -= 'a';
            if (cur->son[c] == nullptr) {
                cur->son[c] = new Node();
            }
            cur = cur->son[c];
        }
        cur->end = true;
    }

    bool search(string word) { return find(word) == 2; }

    bool startsWith(string prefix) { return find(prefix); }
};

/**
 * Your Trie object will be instantiated and called as such:
 * Trie* obj = new Trie();
 * obj->insert(word);
 * bool param_2 = obj->search(word);
 * bool param_3 = obj->startsWith(prefix);
 */
```

## 01 Trie（异或 Trie）

例题：[P4551 最长异或路径](https://www.luogu.com.cn/problem/P4551)。

所谓 01 Trie，就是前面的字符集 $\Sigma=\{0,1\}$ 的 Trie。我们可以利用 01 Trie 快速解决一些异或相关的问题。**01 Trie 的本质就是线段树**，因此能够线段树维护的信息我们也可以用 01 Trie 维护。

以下面这道题为例。

>[!example]  LC 3845
>给你一个非负整数数组 `nums` 和一个整数 `k`。
>
>你需要选择 `nums` 的一个 **子数组**，使得该子数组中元素的 **最大值** 与 **最小值** 之间的差值不超过 `k`。这个子数组的 **值** 定义为子数组中所有元素按位异或（XOR）的结果。
>
>返回一个整数，表示所选子数组可能获得的 **最大值** 。
>
>**子数组** 是数组中任意连续、**非空** 的元素序列。
>
>数据范围：$|nums|\leq 4\times 10^4,\max \{nums\}<2^{15}$。

简要思考，我们很容易去维护第一个部分，难点在于怎么快速查找最大异或值。

我们可以利用 01 trie 解决这个问题。很容易发现如果我们对 $pre_k:=\bigoplus_{i=1}^ka_i$（这里认为是1-indexed）进行异或的话，我们就知道 $[l,r]$ 的异或相当于 $pre_r\oplus pre_{l-1}$。这是十分方便的，这说明我们可以把 $pre$ 塞进一个 01 trie，然后尝试每一步贪心取使得结果最大的。这个贪心是正确的，因为当前这一步的影响 $2^k$ 是大于等于之后所有的影响之和 $1+\cdots+2^{k-1}$ 的。

这样我们就在 $O(n\log V)$ 的时间复杂度下解决了这道题，其中 $V=\max a_i$，还是相当优秀的。

```cpp
const int MAXN = 4e4 + 5;

class Trie {
    struct Node {
        int ch[2]{};
        int count = 0;
    } trie[MAXN * 20];
    int root = 1, tot = 1;

public:
    void insert(int val, int d) {
        int cur = root;
        for (int i = 15; i >= 0; i--) {
            int t = (val >> i) & 1;
            if (!trie[cur].ch[t])
                trie[cur].ch[t] = ++tot;
            cur = trie[cur].ch[t];
            trie[cur].count += d;
        }
    }
    int query(int val) {
        int cur = root;
        int res = 0;
        for (int i = 15; i >= 0; i--) {
            int t = (val >> i) & 1, tt = t ^ 1;
            if (trie[cur].ch[tt] && trie[trie[cur].ch[tt]].count > 0) {
                res |= (1 << i);
                cur = trie[cur].ch[tt];
            } else
                cur = trie[cur].ch[t];
        }
        return res;
    }
};

class Solution {
public:
    int maxXor(vector<int>& nums, int k) {
        int n = nums.size();
        vector<int> pre(n + 1, 0);
        for (int i = 0; i < n; i++)
            pre[i + 1] = pre[i] ^ nums[i];

        Trie trie;
        deque<int> dqmax, dqmin;
        int l = 0, ans = 0;
        for (int r = 0; r < n; r++) {
            while (dqmax.size() && nums[dqmax.back()] < nums[r])
                dqmax.pop_back();
            while (dqmin.size() && nums[dqmin.back()] > nums[r])
                dqmin.pop_back();
            dqmax.push_back(r), dqmin.push_back(r);

            trie.insert(pre[r], 1);
            while (nums[dqmax.front()] - nums[dqmin.front()] > k) {
                trie.insert(pre[l], -1);
                l++;
                if (dqmax.front() < l)
                    dqmax.pop_front();
                if (dqmin.front() < l)
                    dqmin.pop_front();
            }
            ans = max(ans, trie.query(pre[r + 1]));
        }
        return ans;
    }
};
```
