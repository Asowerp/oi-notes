## 单调栈

### 用途

查询第一个大于 / 小于（等于）某个给定值的位置 / 数值。

### 思路

我们考虑这样的维护思路。以查询队列前方第一个小于某个给定值为例。

假设现在栈顶是 $a[i]$，一旦后面出现了比 $a[i]$ 还小的 $a[j]$，那就可以把 $a[i]$ 踢掉，因为 $a[j]$ 必然是第一个被扫到的。

>[!example]
>设计一个算法收集某些股票的每日报价，并返回该股票当日价格的 **跨度** 。
>
>当日股票价格的 **跨度** 被定义为股票价格小于或等于今天价格的最大连续日数（从今天开始往回数，包括今天）。
>
> - 例如，如果未来 7 天股票的价格是 `[100,80,60,70,60,75,85]`，那么股票跨度将是 `[1,1,1,2,1,4,6]` 。
> 
> 实现 `StockSpanner` 类：
>- `StockSpanner()` 初始化类对象。
>- `int next(int price)` 给出今天的股价 `price` ，返回该股票当日价格的 **跨度** 。

本题亦即查询第一个大于 `price` 的位置。

```cpp
class StockSpanner {
    stack<pair<int, int>> s;
    int cur_day;
public:
    StockSpanner() { cur_day = 0; }
    int next(int price) {
        while (!s.empty() && price >= s.top().second) {
            s.pop(); // 这时第一个被扫到的一定是当日的 price。
        }
        cur_day++;
        int ans = cur_day;
        if (!s.empty())
            ans -= s.top().first;
        s.push({cur_day, price});
        return ans;
    }
};
```



## 单调队列

单调队列的基本思想是“如果一个人比你小还比你强，那么你就被 T 了”。

>[!example]
>给定 $n$ 个数，求任意连续 $k$ 个数的最大、最小值。

顾名思义，单调队列的重点分为“单调”和“队列”。
 - “单调”指的是元素的“规律”——递增（或递减）。
 - “队列”指的是元素只能从队头和队尾进行操作。

假定 $k=3$，序列如下

```
1 3 -1 -3 5 3 6 7
```

因为我们始终要维护队列保证其 **递增** 的特点，所以会有如下的事情发生：

| 操作                              | 队列状态      |
| :-----------------------------: | :-------: |
| 1 入队                            | `{1}`     |
| 3 比 1 大，3 入队                    | `{1 3}`   |
| -1 比队列中所有元素小，所以清空队列 -1 入队       | `{-1}`    |
| -3 比队列中所有元素小，所以清空队列 -3 入队       | `{-3}`    |
| 5 比 -3 大，直接入队                   | `{-3 5}`  |
| 3 比 5 小，5 出队，3 入队               | `{-3 3}`  |
| -3 已经在窗体外，所以 -3 出队；6 比 3 大，6 入队 | `{3 6}`   |
| 7 比 6 大，7 入队                    | `{3 6 7}` |
单调队列有多中实现方式。
 - 可以记录两个数组 q 和 p，分别代表取值集合与位置集合；
 - 可以仅仅维护位置集合；
 - 也可以利用 deque 并自己维护一个结构体同时维护取值和位置。

理论上手写比用 deque 快但是我懒了。以求最小值为例。

```cpp
deque<Node> dq;
while(!dq.empty() && dq.back().val >= a[i]) dq.pop_back();
while(!dq.empty() && dq.front().pos <= i - k) dq.pop_front();
dq.push_back({a[i], i});
```

这样队首元素一直都是最小值。
