#树状数组

树状数组（Fenwick Tree）本质上就是高级一点的区间前缀和。它把 $n$ 拆成若干个 2 的次幂的和从而化简计算。基础代码如下。

```cpp
inline int lowbit(int x) {
    return x & (-x);
}

void add(int pos, int x) {
    for (; pos < (int) bit.size(); pos += lowbit(pos)) bit[pos] += x;
}

int ask(int pos) {
    int res = 0;
    for (; pos > 0; pos -= lowbit(pos)) res += bit[pos];
    return res;
}
```

其中 `ask` 返回前缀和。

如果想写 0-indexed，以及 `query(r)` 返回 $[0,r)$ 的，可以这么写：

```cpp
class Fenwick {
    int n;
    vector<int> data;
    public:
    Fenwick(int _n) : n(_n), data(_n,0) {}
    void add(int pos,int val) {
        for(;pos<n;pos|=pos+1) data[pos]+=val;
    }
    int query(int pos) { // 返回 [0,pos)
        int ret=0;
        for(;pos>0;pos&=pos-1) ret+=data[pos-1];
        return ret;
    }
    int query(int l,int r) { // 返回 [l,r]
        return query(r+1)-query(l);
    }
};
```

>[!example] T27256 当前队列中位数
>中位数是有序序列最中间的那个数。如果序列的长度是偶数，则没有最中间的数；此时中位数是最中间的两个数的平均数。
>
>现在，假设我有一个空的 list，我会对他进行以下三种操作：
>1. 在 list 最后添加一个数：`add x`
>2. 删除当前 list 的第一个数：`del`
>3. 查询当前 list 的中位数：`query`
>
>针对每次 `query`，输出当前的中位数。

其实这道题是一个 `multiset` 板子题，但我忘了 `multiset` 是自动排序的，这太谔谔了。

但是显然如果我们用树状数组的话肯定要离线操作。

这种询问第 k 小的一般都可以转化成树状数组来做。我们来讲一下询问第 k 小的写法。

第 k 小的写法长这样。

```cpp
int kth(int k) {
    int pos = 0;
    for (int i = 1 << (31 - __builtin_clz((int) bit.size() - 1)); i;
         i >>= 1) {
        if (pos + i < (int) bit.size() && bit[pos + i] < k) {
            k -= bit[pos + i];
            pos += i;
        }
    }
    return pos + 1;
}
```

**只有在 `bit` 存储的是频数的情况下才能这么写！！** 也就是说这个 bit 数组是所谓的权值树状数组。

`kth(int k)` 返回的就是第 k 小在原数组中的位置（离散化之后的编号）。为什么呢？
- **`i` 的初始值**：
    - `__builtin_clz(x)` 返回 `x` 的二进制前导零的个数。  
    - `31 - __builtin_clz(bit.size() - 1)` 计算 `bit.size() - 1` 的最高有效位位置。
    - `1 << MSB` 得到最大的 `i`。
- **循环条件**：
    - `i` 每次右移 `1` 位（相当于 `i /= 2`），直到 `i = 0` 时终止。
- 后面其实和树状数组差不多了，注意到 `bit` 掌控的是区间和，所以每次 k 都要减去 `bit[pos + i]`。
	- **检查当前区间的和是否 `< k`**：
	    - `bit[pos + i]` 存储的是 `[pos + 1, pos + i]` 的某种统计值（如频率和）。
	    - 如果 `bit[pos + i] < k`，说明第 `k` 小的数不在当前区间，调整 `k` 和 `pos`：    
	        - `k -= bit[pos + i]`：减去当前区间的贡献。
	        - `pos += i`：跳到下一个区间。
	- **否则**：
	    - 第 `k` 小的数在当前区间，继续缩小范围（`i >>= 1`）。
- 最后会停在 `k - 1` 小的位子（因为我们上面的策略保证了是 `k-1` 的末尾——只要比 `k` 严格小就往前跳），所以 `return pos + 1`。

这题要记得加个离散化，不然复杂度飞起来了。

`vals.erase(unique(vals.begin(), vals.end()), vals.end());` 可以直接去重。原理是 `unique` 把所有重复元素移到了不重复集合的末尾并返回该位置的迭代器，我们直接 `erase` 一下就去掉了所有重复元素。

下面代码复杂度 $O(n\log n)$，显然能过。

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, tot, l;
vector<int> a, bit, vals;

inline int lowbit(int x) {
    return x & (-x);
}

void add(int pos, int x) {
    for (; pos < (int) bit.size(); pos += lowbit(pos)) bit[pos] += x;
}

int ask(int pos) {
    int res = 0;
    for (; pos > 0; pos -= lowbit(pos)) res += bit[pos];
    return res;
}

int kth(int k) {
    int pos = 0;
    for (int i = 1 << (31 - __builtin_clz((int) bit.size() - 1)); i;
         i >>= 1) {
        if (pos + i < (int) bit.size() && bit[pos + i] < k) {
            k -= bit[pos + i];
            pos += i;
        }
    }
    return pos + 1;
}

double query(int n) {
    if (n % 2)
        return vals[kth(n / 2 + 1) - 1];
    else
        return (vals[kth(n / 2) - 1] + vals[kth(n / 2 + 1) - 1]) /
               2.0;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n;
    vector<pair<string, int>> ops(n);
    for (int i = 0; i < n; i++) {
        string op;
        int x = 0;
        cin >> op;
        if (op == "add") {
            cin >> x;
            vals.push_back(x);
        }
        ops[i] = {op, x};
    }

    sort(vals.begin(), vals.end());
    vals.erase(unique(vals.begin(), vals.end()), vals.end());
    bit.assign(vals.size() + 2, 0);
    a.assign(n + 2, 0);

    l = 1;
    for (int i = 0; i < n; i++) {
        auto [op, x] = ops[i];
        if (op == "add") {
            int id = lower_bound(vals.begin(), vals.end(), x) -
                     vals.begin() + 1;
            a[++tot] = id;
            add(id, 1);
        } else if (op == "del") {
            add(a[l++], -1);
        } else {
            cout << query(tot - l + 1) << '\n';
        }
    }
}
```
