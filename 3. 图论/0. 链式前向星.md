# 链式前向星

链式前向星是一种非常省空间的（$O(n+m)$）的存图方法。

在存图过程中，我们要用到三个数组，$head$，$next$ 和 $to$，以及一个辅助变量 $cnt$。$head$ 表示“头部边”，也就是开始遍历的时候从哪条边开始。$next$ 表示“下一条边的编号”，也就是遍历某条边之后下一个要遍历的边的编号。$to$ 表示这条边的终点。

工作原理如下：
 - 初始状态，全部置 0，$cnt=1$。
 - 读入 $(u,v)$ 是一条边，编号为 $cnt$，$cnt\leftarrow cnt+1$。
	 - 修改 $next_{cnt}=head_u$，表示遍历完这一条边之后，下一个要遍历的是 $u$ 的头部边。
	 - 修改 $head_u=cnt$，表示遍历要从 $cnt$ 开始。
	 - 修改 $to_{cnt}=v$，表示这条边的重点是 $v$。

如果我要遍历一个点 $u$ 的边，我就从 $head_u$ 出发，设当次编号为 $k$，重复赋值 $k\leftarrow next_k$，直到 $k=0$，这表示没有下一条边了。

因此，每个点只用维护自己头部边的编号，每个边只要维护自己要到达的点和下一条边，空间复杂度是 $O(n+m)$。

```cpp
int head[MAXN], nxt[MAXM], to[MAXM], cnt;

void init() {
    memset(head, 0, sizeof(head));
    cnt = 0;
}

void addEdge(int u, int v) {
    cnt++;
    nxt[cnt] = head[u];
    to[cnt] = v;
    head[u] = cnt;
}
```
