什么是反演？  

简单理解一下，若能从 $f$ 推出 $g$，则称之为**正演**。那么将其倒过来，用 $g$ 反推出 $f$ 就叫做**反演**了。

从题目条件出发进行简单处理得到一个式子，答案也可以通过正演得到这个式子，那么这时候就可以利用反演通过这个式子推出答案。这种逆向思维在解题中能够帮助我们更好地理解问题的结构，找到更简便的解法。

## 二项式反演

二项式反演常用于解决**至多**（**至少**）和**恰好**之间的转换，常用于统计方案数。  一般来说，知道**恰好**后是能较为容易的推出**至多**的，我们这里先写一个式子：
$$g(n)=\sum_{i=0}^n\binom ni f(i)$$
这里，$g(n)$ 表示**至多**选 $n$ 个的方案数，$f(i)$ 表示**恰好**选 $i$ 个的方案数。显然至多就是要把所有 $0$ 到 $n$ 的恰好方案加上去。

在题目中，一般 $g$ 是较为好求的。那么接下来我们就要解决：**如何从 $g$ 反推回 $f$**？

考虑容斥原理。
$$\left| \bigcup_{i=1}^n A_i \right| = \sum_i |A_i| - \sum_{i,j} |A_i \cap A_j| + \cdots + (-1)^{n+1} \left| \bigcap_{i=1}^n A_i \right|$$
补集的交等于并的补集。记 $B_i=A_i^c$，于是我们改写一下：
$$\left| \bigcap_{i=1}^n B_i \right| = |U| - \sum_i |A_i| + \sum_{i,j} |A_i \cap A_j| - \cdots + (-1)^n \left| \bigcap_{i=1}^n A_i \right|$$
而补集的补集变成了原集，于是将 $B_i$ 和 $A_i$ 互换有：
$$\left| \bigcap_{i=1}^n A_i \right| = |U| - \sum_i |B_i| + \sum_{i,j} |B_i \cap B_j| - \cdots + (-1)^n \left| \bigcap_{i=1}^n B_i \right|$$

这两者只是 $A, B$ 定义上的区别，系数完全一样，能不能把它们联系起来呢？  
记 $f(i)$ 表示 $i$ 个 $A$ 集合的交集大小，$g(i)$ 表示 $i$ 个 $B$ 集合的交集大小。这里是一种特殊情况：即任意的 $i$ 个集合 $A$ 交起来的大小相等，但同样也满足上面那个式子。

则有：
$$f(n) = \sum_{i=0}^n (-1)^i \binom{n}{i} g(i) \iff g(n) = \sum_{i=0}^n (-1)^i \binom{n}{n-i} f(i)$$

看起来非常的对称。需要注意的是，其中的 $f(0) = g(0) = |U|$，这样就能与上述式子相符了。

**注意**：在真正做题的时候，我们只知道 $f, g$，而无需考虑上面证明时提到的集合。换句话说，这只是种较为简单易懂的证明方式，**二项式反演虽然形式上和多步容斥极为相似，但它们并不等价**。

因此我们得到了下面两个常用形式。

>[!theorem] 二项式反演
>1. 至多与恰好的转换：
>$$g(n) = \sum_{i=0}^n \binom{n}{i} f(i) \iff f(n) = \sum_{i=0}^n (-1)^{n-i} \binom{n}{i} g(i)$$
>2. 至少与恰好的转换（常用）：
>$$g(n) = \sum_{i=n}^N \binom{i}{n} f(i) \iff f(n) = \sum_{i=n}^N (-1)^{i-n} \binom{i}{n} g(i)$$

值得注意的是，第二个式子里面的 $N$ 应该是 $+\infty$，然而题目通常限制了之上的应该是 0，所以 $N$ 就足矣。

>[!example] P10596
>一个有 $N$ 个元素的集合有 $2^N$ 个不同子集（包含空集），现在要在这 $2^N$ 个集合中取出若干集合（至少一个），使得它们的交集的元素个数为 $K$，求取法的方案数，答案模 $10^9+7$。

模板。我们直接拍上去一个转化二。令 $g(k)$ 表示交集元素个数至少为 $k$ 的方法数，那么我们先钦定 $k$ 个元素作为交集，这是 $\binom nk$ 的，然后总共有 $2^{n-k}$ 个集合满足。然后可以随便取，所以 $g(k)=\binom nk\cdot (2^{2^{n-k}}-1)$。然后利用 $f(n) = \sum_{i=n}^N (-1)^{i-n} \binom{i}{n} g(i)$，就可以得到一个优秀的 $O(n\log n)$ 做法。但是要注意我们次幂上的 $2^{n-k}$ 要对 `mod-1` 取模（考虑 Fermat 小定理）

```cpp
#include <bits/stdc++.h>

using namespace std;

#define all(x) begin(x), end(x)
#define sz(x) (int) (x).size()

using ll = long long;
using pii = pair<int, int>;
using vi = vector<int>;
using usi = unordered_set<int>;

inline int read() {
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + (ch - '0');
        ch = getchar();
    }
    return x * f;
}

const int MOD = 1e9 + 7;

int qpow(int base, int pow, int mod = MOD) {
    int ret = 1;
    while (pow) {
        if (pow & 1) ret = (1ll * ret * base) % mod;
        base = (1ll * base * base) % mod, pow >>= 1;
    }
    return ret;
}

const int MAXN = 1e6 + 5;

inline int inv(int x) { return qpow(x, MOD - 2); }

int fac[MAXN], invfac[MAXN], pow2[MAXN];

void init(int n) {
    fac[0] = 1, invfac[0] = 1, pow2[0] = 1;
    for (int i = 1; i <= n; i++) {
        fac[i] = (1ll * fac[i - 1] * i) % MOD;
        invfac[i] = inv(fac[i]);
        pow2[i] = (1ll * pow2[i - 1] * 2) % MOD;
    }
}

inline int nCr(int n, int r) {
    return (1ll * ((1ll * fac[n] * invfac[n - r]) % MOD) *
            invfac[r]) %
           MOD;
}

int main() {
    cin.tie(nullptr)->sync_with_stdio(false);
    int n = read(), k = read();
    init(n);
    ll ans = 0;
    for (int i = k; i <= n; i++) {
        ll c1 = (1ll * nCr(n, i) * nCr(i, k)) % MOD;
        ll c2 =
            (qpow(2, qpow(2, n - i, MOD - 1), MOD) - 1 + MOD) % MOD;
        ll c = (c1 * c2) % MOD;

        if ((i - k) % 2 == 1) {
            ans = (ans - c + MOD) % MOD;
        } else {
            ans = (ans + c) % MOD;
        }
    }
    cout << ans << endl;
    return 0;
}
```

## Mobius 反演（倍数反演）

以下面这道题为例。

> [!example] LC 3671 
> 给你一个长度为 `n` 的整数数组 `nums`。
>
>对于每个 **正整数** `g`，定义 `g` 的 **美丽值** 为 `g` 与 `nums` 中符合要求的子序列数量的乘积，子序列需要 **严格递增** 且最大公约数（GCD）恰好为 `g` 。
>
>请返回所有正整数 `g` 的 **美丽值** 之和。
>
>由于答案可能非常大，请返回结果对 `1e9 + 7` 取模后的值。
