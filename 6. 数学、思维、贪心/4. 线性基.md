#线性基 

一般来讲，线性基都指的是 $\newcommand{\FF}{\mathbb F}\FF_2$ 上的线性基。当然也有一般线性空间上的线性基。我们通过下面这道题来引入。

>[!example] P3812
>给定 $n$ 个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。
>
>保证值域 $V\leq 2^{63}$。

很显然，在既然出现了异或，那肯定是期望在二进制下考虑。每个数都可以等价于 $\FF_2^n$ 中的一个元素。所以现在我们只用找到这组向量在 $\FF_2^n$ 下的一组基就好了。这样的构造有以下两种方法。两者的构造复杂度都是 $O(n\log V)$。

为叙述方便，称二进制表示中为 1 的位为「置位」。
## 贪心法

贪心法的流程简单总结如下。
1. 我们构造数组 $p_i$，其中 $p_i$ 表示最高置位在第 $i$ 位的数。若不存在则为 0。
2. 依次插入数 $x$，从高到低扫。
	1. 如果 $x$ 的第 $i$ 位为 1 且 $p_i=0$，则 $p_i\gets x$，并退出循环。
	2. 反之，令 $x\gets x\oplus p_i$。
	3. 重复上述操作，直到 $x=0$，这说明 $x$ 可以被表出。

上面这个算法的正确性是显然的。因为每一次操作我们相当于进行了一次线性组合，而这并不会改变基的性质。

至于查询，则同理，我们初始 $s=0$，不断令 $s\gets\max\{s,s\oplus p_i\}$ 即可。

每一次操作我们都是 $O(\log V)$ 的，因此总的时间复杂度是 $O(n\log V)$。

```cpp
#include <bits/stdc++.h>

using namespace std;

#define all(x) begin(x), end(x)
#define sz(x) (int) (x).size()

using i64 = long long;
using u64 = unsigned long long;
using i128 = __int128_t;
using u128 = __uint128_t;
using pii = pair<int, int>;
using vi = vector<int>;
using usi = unordered_set<int>;

class Basis {
    vector<i64> p;

  public:
    Basis() : p(64, 0) {}
    void insert(i64 x) {
        for (int i = 50; i >= 0 && x; i--) {
            if ((x >> i) & 1) {
                if (!p[i]) {
                    p[i] = x;
                    break;
                } else {
                    x ^= p[i];
                }
            }
        }
    }
    i64 get() {
        i64 ret = 0;
        for (int i = 50; i >= 0; i--) {
            if (p[i]) {
                ret = max(ret, ret ^ p[i]);
            }
        }
        return ret;
    }
};

int main() {
    cin.tie(nullptr)->sync_with_stdio(false);
    int n;
    cin >> n;
    Basis b;
    for (int i = 0; i < n; i++) {
        i64 x;
        cin >> x;
        b.insert(x);
    }
    cout << b.get() << '\n';
    return 0;
}
```

我们也可以实现查询某个数是否能被异或。和插入是完全类似的逻辑。
