#拓展欧几里得算法

熟知下面的定理。

>[!theorem] Bezout 定理
>方程 $ax+by=c$ 有整数解的充要条件是 $\gcd(a,b)\mid c$。

因此如果方程 $ax+by=c$ 有整数解，那么 $bx+(b\mod a)y=c$ 亦有整数解。这是由辗转相除法保证的。

我们从特殊情况开始考虑。命 $d=\gcd(a,b)$，反复进行上面的操作，到最后，会得到 $dx+0y=d$ 这个方程。其显然有 $(x,y)=(1,0)$ 这一组解。

现在假设我们已经求得了 $bx+(b\mod a)y=d$ 的一组解 $(x',y')$，我们要怎么还原成原先的形式呢？直接恒等变形。

$$
\begin{aligned}
bx'+(a\mod b)y'&=d,\\
bx'+\left(a-\left\lfloor\frac ab\right\rfloor\right)y'&=d,\\
ay'+b\left(x'-\left\lfloor\frac ab\right\rfloor y'\right)&=d.
\end{aligned}
$$
也就是说，我们递归求解，然后命 $x\leftarrow y'$，$y\leftarrow x'-\left\lfloor\dfrac ab\right\rfloor y'$，到最后我们就得到了 $ax+by=d$ 的一组解 $(x_0,y_0)$。

也就是说，原方程的一组解是 $\left(x_0\cdot \frac cd, y_0\cdot \frac cd\right)$。

那么，怎么找到最小的一组正整数解呢？可以用数学归纳法证明，对于 $ax+by=d$ 来说，在上述递推过程中的最后一步命 $(x,y)=(1,0)$，最后得到的解满足 $|x|\leq b, |y|\leq a$，并且 $|x|,|y|$ 同时取到最小值。

但是对更加一般的 $c$，我们不一定有这样优美的性质。在下面的过程中，我们规定 $\mod m$ 运算的结果在 $\{0,1,\cdots, m-1\}$ 之间。

有下面定理成立：

>[!命题]
>若 $\gcd(a,b)=1$ , 且 $x_0​$， $y_0$​ 是 $ax+by=c$ 的一组解，则方程任一解可表示为：$x=x_0​+bt$， $y=y_0​−at$，且对于任一整数 $t$ 皆成立。

下设 $(x_1,y_1), (x_2,y_2)$ 都是 $ax+by=c$ 的解。则
$$
a(x_1-x_2)+b(y_1-y_2)=0.
$$
同时除以 $d=\gcd(a,b)$，则
$$
\frac ad(x_1-x_2)+\frac bd(y_1-y_2)=0.
$$
因此 $x_1-x_2=k\cdot \frac bd$。$x = x_0+\frac bd\cdot k$。因此 $x$ 的最小正整数解是 $x_0\mod \frac bd$。同理 $y$ 的最小正整数解是 $y_0\mod \frac ad$。

代码如下。（[P5656 【模板】二元一次不定方程 (exgcd) - 洛谷](https://www.luogu.com.cn/problem/P5656)）

```cpp
#include <bits/stdc++.h>

using namespace std;

inline int mod(int x, int y) {
    int t = (x % y + y) % y;
    return t ? t : y;
}

void exgcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1, y = 0;
        return;
    }
    int x0 = 0, y0 = 0;
    exgcd(b, a % b, x0, y0);
    x = y0, y = x0 - (a / b) * y0;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);
    int t;
    cin >> t;
    while (t--) {
        int a, b, c;
        cin >> a >> b >> c;
        int g = __gcd(a, b);
        if (c % g) {
            cout << -1 << endl;
            continue;
        }
        int x = 0, y = 0, c0 = c / g, b0 = b / g, a0 = a / g;
        exgcd(a, b, x, y);
        x *= c0, y *= c0;
        int xmin = mod(x, b0), ymin = mod(y, a0),
            xmax = (c - ymin * b) / a, ymax = (c - xmin * a) / b;
        if (xmin > 0 && ymax > 0) {
            int cnt = (xmax - xmin) / b0 + 1;
            cout << cnt << ' ' << xmin << ' ' << ymin << ' ' << xmax
                 << ' ' << ymax << endl;
        } else {
            cout << xmin << ' ' << ymin << endl;
        }
    }
    return 0;
}
```
