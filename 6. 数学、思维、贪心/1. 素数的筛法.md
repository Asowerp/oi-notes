#筛法

## 埃氏筛

埃氏筛的基本思想是，如果一个质数已经被判定了，那么它的若干倍都是和数。

于是很自然地写出下面这个筛法。

```cpp
void seive(int x) {
	for (int i = 2; i <= x; i++) {
		if (!vis[i]) {
			for(int j = i * i; j <= x; j += i) {
				// 小于 i * i 的已经被前面的筛过了。
				vis[j] = true;
			}
		}
	}
}
```

这个算法的时间复杂度是 $O(n\log\log n)$。

## 欧拉筛

欧拉筛在埃氏筛的基础上进行了改进，使得时间复杂度降为 $O(n)$。

```cpp
vector<int> pri; 
bool not_prime[N]; 
void pre(int n) { 
	for (int i = 2; i <= n; ++i) { 
		if (!not_prime[i]) { pri.push_back(i); } 
		for (int pri_j : pri) { 
			if (i * pri_j > n) break; 
			not_prime[i * pri_j] = true; 
			if (i % pri_j == 0) { 
				// i % pri_j == 0 
				// 换言之，i 之前被 pri_j 筛过了 
				// 由于 pri 里面质数是从小到大的，所以 i 乘上其他的质数的结果一定被 
				// pri_j 的倍数筛掉，就不需要在这里先筛一次，所以这里直接 break 
				// 掉就好了 
				break; 
			} 
		} 
	} 
}
```

如果要进一步压缩空间复杂度，`not_prime[N]` 可以换成 `bitset<N> not_prime`。
