# 状态 DP

>[!example] CF545C
>有 $n$ 棵树按坐标升序排列，每棵树可以砍倒向左或向右占据一定区间，也可以不砍。要求倒下树的区间不能重叠且不能压到未砍树的坐标点。求最多能砍倒多少棵树。
>$n\leq 2\times 10^5$.

显然的状态 DP。用 $f[i][0/1/2]$ 表示到达第 $i$ 棵树，不砍 / 向左倒 / 向右倒最多能砍多少棵。然后就是想转移方程了。这个转移方程是需要注意的。具体可以看代码。

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> pos(n + 1, 0), h(n + 1, 0);
    for (int i = 1; i <= n; i++) { cin >> pos[i] >> h[i]; }
    vector<vector<int>> dp(n + 1, vector<int>(3, 0));
    for (int i = 1; i <= n; i++) {
        dp[i][0] = max({dp[i - 1][0], dp[i - 1][1], dp[i - 1][2]});
        if (i == 1 || pos[i - 1] < pos[i] - h[i])
            dp[i][1] = max({dp[i - 1][0], dp[i - 1][1]}) + 1;
        if (i == 1 || pos[i - 1] + h[i - 1] < pos[i] - h[i])
            dp[i][1] = max(dp[i][1], dp[i - 1][2] + 1);
        if (i != n) {
            if (pos[i + 1] > pos[i] + h[i]) {
                dp[i][2] = max({dp[i - 1][0], dp[i - 1][1]}) + 1;
                if (pos[i - 1] + h[i - 1] < pos[i])
                    dp[i][2] = max(dp[i][2], dp[i - 1][2] + 1);
            }
        } else {
            dp[i][2] = max({dp[i - 1][0], dp[i - 1][1]}) + 1;
            if (pos[i - 1] + h[i - 1] < pos[i])
                dp[i][2] = max(dp[i][2], dp[i - 1][2] + 1);
        }
    }
    cout << max({dp[n][0], dp[n][1], dp[n][2]}) << endl;
    return 0;
}

```
