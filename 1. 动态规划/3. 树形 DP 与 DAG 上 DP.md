## 树形 DP

**树形 DP** 是一种在固定一个根节点的树上，利用 **子树的信息** 来进行计算的动态规划方法。

我们定义：
$$
dp[v] := 以顶点\ v\ 为根的子树所对应的某个值
$$
如果一棵树有 $N$ 个节点，那么它就有 $N$ 个子树，因此需要计算 $N$ 个 dp 值。

计算方法通常是：使用 DFS 从每个子树开始，**自底向上** 地计算 dp。

由于 DFS 过程中如果有环就会导致计算无法结束，因此树可以看作是一个有根的 **DAG（有向无环图）** 来处理。

>[!例子]
>某大学有 $N$ 个职员，编号为 $1 \sim N$。他们之间有从属关系，也就是说他们的关系就像一棵以校长为根的树，父结点就是子结点的直接上司。现在有个周年庆宴会，宴会每邀请来一个职员都会增加一定的快乐指数 $a_i$，但是呢，如果某个职员的直接上司来参加舞会了，那么这个职员就无论如何也不肯来参加舞会了。所以，请你编程计算，邀请哪些职员可以使快乐指数最大，求最大的快乐指数。

我们记 $dp[i][x]$ 表示以 $i$ 为根的子树的最优解，$x=0$ 表示 $i$ 不参加，$x=1$ 反之。则
$$
\begin{aligned}
dp[i][0]&=\sum_{j\in\operatorname{son}(i)}\max(dp[j][1],dp[j][0]);\\
dp[i][1]&=\sum_{j\in\operatorname{son}(i)} dp[j][0]+a[i].
\end{aligned}
$$
这样我们就可以写出下面这个 DFS
```cpp
void dfs(int x) {
	if (vis[x]) return;
	vis[x] = 1;
	dp[x][1] = a[x];
	for (auto u : edge[x]) {
		dfs(u); // 这样保证了子树先枚举，从而自下往上。
		dp[x][0] += max(dp[u][1], dp[u][0]);
		dp[x][1] += dp[u][0];
	}
}
```

通常，树形 DP 状态一般都为当前节点的最优解。先 DFS 遍历子树的所有最优解，然后向上传递给子树的父节点来转移，最终根节点的值即为所求的最优解。

## DAG 上 DP

DAG 是指有向无环图。

>[!example]  at_dp_g Longest Path
>求**有向无环图**上的最长路长度。

容易写出下面的代码。

```cpp
#include <bits/stdc++.h>

using namespace std;

inline int read() {
    int f = 1, x = 0;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + (ch - '0');
        ch = getchar();
    }
    return x * f;
}

template <typename T> inline void read(T &x) {
    int f = 1;
    x = 0;
    char ch = getchar();

    while (ch < '0' || ch > '9') {
        if (ch == '-') f *= -1;

        ch = getchar();
    }

    while (ch <= '9' && ch >= '0') {
        x = x * 10 + ch - '0';
        ch = getchar();
    }

    x *= f;
}

template <typename T, typename... Args>
inline void read(T &x, Args &...args) {
    read(x), read(args...);
}

const int MAXN = 1e5 + 5;

int n, m;
vector<int> e[MAXN];
int ans;
int dp[MAXN];

void dfs(int u) {
    if (dp[u]) return;
    if (e[u].empty()) return;
    for (auto v : e[u]) {
        dfs(v);
        dp[u] = max(dp[u], dp[v] + 1);
        ans = max(dp[u], ans);
    }
}

int main() {
    read(n, m);
    for (int i = 1; i <= m; i++) {
        int x, y;
        read(x, y);
        e[x].emplace_back(y);
    }
    for (int i = 1; i <= n; i++) {
        dfs(i);
    }
    cout << ans << endl;
    return 0;
}
```
