#DP #动态规划

**二维问题一定要想着转化为一维！！！**

DP 的重点就是找递推关系。

>[!example] T27373 最大整数
>假设有 $n$ 个正整数，要求从中选取几个组成一个位数不超过 $m$ 的新正整数，现需要求出最大可能数值是多少。
>
>比如有 4 个数，993，923，3817，3807，组成新数不超过 7 位，那么新数最大值为 9933817。

这道题肯定考虑贪心 + dp。

思考一下策略。我们希望拼接意义下的字典序最大的放前面，也就是说我们排序的 cmp 函数要这么写：

```cpp
auto cmp = [](string a, string b) { return a + b > b + a; }
```

下一步就是 dp。我们考虑下面这个定义：
$$
dp[i][j]:=\text{使用了前}\ i\ \text{个字符串，位数为}\ j\ \text{的最大整数}.
$$
这样我们就可以写出下面这个转移方程：
$$
dp[i][j]=\max(dp[i-1][j],dp[i-1][j-\operatorname{digit}[i]]+s[i]).
$$
这里的加是在字符串意义下的。我们最后要输出最大的，就是要最大的位数（因为有可能 $m$ 无解）。于是可以写出下面的 AC 代码。

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, m;
string s[1005];
string dp[1005][205];
string ans;

int main() {
    cin >> m >> n;
    for (int i = 1; i <= n; i++) cin >> s[i];
    sort(s + 1, s + 1 + n,
         [](string a, string b) { return a + b > b + a; });
    for (int i = 1; i <= n; i++) {
        for (int j = 0; j <= m; j++) {
            dp[i][j] = dp[i - 1][j];
            if (dp[i - 1][j - s[i].size()].size() == j - s[i].size())
                dp[i][j] = max(dp[i - 1][j - s[i].size()] + s[i],
                               dp[i - 1][j]);
        }
    }

    for (int i = m; i > 0; i--) {
        if (!dp[n][i].empty()) {
            ans = dp[n][i];
            break;
        }
    }
    cout << ans << endl;
    return 0;
}
```

>[!example] P1719 最大加权矩形 / M02766 最大子矩阵
>现有一个 $n\times n\ (1\leq n\leq 120)$ 矩阵。要求矩阵中最大加权矩形，即矩阵的每一个元素都有一权值，权值定义在整数集上。从中找一矩形，矩形大小无限制，是其中包含的所有元素的和最大。矩阵的每个元素属于 $[-127,127]$ 。

这道题用的就是二维转一维的想法。

事实上注意到 $n\leq 120$，所以 $O(n^4)$ 的前缀和暴力也能过。

我们考虑枚举两行之间（枚举 $i,j$ 行），这样问题就转化为 $i,j$ 行之间的数的最大连续子列和（按列）。

所以就有下面的代码。注意最大值要时刻更新。

```cpp
#include <bits/stdc++.h>

using namespace std;

int n;
int a[200][200], s[200][200];
int ans = INT_MIN, cur_ans = INT_MIN;

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            cin >> a[i][j];
            s[i][j] = s[i][j - 1] + a[i][j];
        }
    }

    for (int i = 1; i <= n; i++) {
        for (int j = i; j <= n; j++) {
            cur_ans = 0;
            for (int k = 1; k <= n; k++) {
                cur_ans = max(cur_ans + s[k][j] - s[k][i - 1],
                              s[k][j] - s[k][i - 1]);
                ans = max(cur_ans, ans);
            }
        }
    }
    cout << ans << endl;
    return 0;
}
```

>[!example] 鸡蛋的硬度
>有一栋 $n$ 层的楼，你可以进行如下操作：
>- 扔一个鸡蛋。
>- 如果鸡蛋碎了，那么鸡蛋数量 ${}-1$。
>
>问：至少扔几次鸡蛋，才能**保证**找出这种鸡蛋最高在第几层楼不碎？
>注意：有可能从第 $n$ 层扔都不会碎。

本题的难点在于找递推关系。

>[!example] at_dp_f LCS
>给定一个字符串 $s$ 和一个字符串 $t$ ，输出 $s$ 和 $t$ 的最长公共子序列（LCS）。

显然对长度，我们可以找到下面这个定义，
$$
dp[i][j]:=s\ \text{的前}\ i\ \text{个和} \ t\ \text{的前}\ j\ \text{个的 LCS}.
$$
不难写出转移方程为
$$
dp[i][j]=\begin{cases}\max(dp[i-1][j],dp[i][j-1]),& s[i]\ne t[j],\\
\max(dp[i-1][j],dp[i][j-1],dp[i-1][j-1]+1),&s[i]=t[j].\end{cases}
$$
## 合法子序列 DP

计算合法子序列的最长长度、个数、元素和等。

一般定义 $f[x]$ 表示以元素 $x$ 结尾的合法子序列的最长长度/个数/元素和，从子序列的倒数第二个数转移过来。

注意这里的 $x$ 不是下标，是**元素值**。如果 $x$ 不是整数，或者值域范围很大，可以用哈希表代替数组（或更常用：离散化）。
